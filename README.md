# Getting this up and running

1. Install Hugo: ```brew install hugo```
2. Run in dev mode: ```./run-server.sh```

# Deploying

This requires the AWS CLI to be set up and configured. Run ```aws configure``` to do so.

1. run  ```./run-deploy.sh```
